typedef uint16(* pfnVerifySecCB_t) (uint16 connHandle, uint8 id, l2capConnectReq_t *pReq)

Callback function prototype to verify security when a Connection Request is received.

Note
It can be specified as NULL by PSM if there's no security requirements.
Parameters
connHandle- connection handle request was received on
id- identifier matches responses with requests
pReq- received connection request
Returns
L2CAP Connection Response Result Values
