Detailed Description

Parameter Structure to add new Characteristic Description Response SNP_ADD_CHAR_DESC_DECL_RSP.

snpAddCharDescDeclRsp_t::handles will be populated with attributes added based on the header, starting from LSB to MSB.

This packet structure is not used to create a NPI frame
