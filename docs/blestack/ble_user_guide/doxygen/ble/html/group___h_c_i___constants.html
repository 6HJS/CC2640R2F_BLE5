#define HCI_CMD_MIN_LENGTH   4

§ HCI_DATA_MIN_LENGTH

#define HCI_DATA_MIN_LENGTH   5

Minimum length for data packet.

Minimum length for DATA packet is 1+2+2 | Packet Type (1) | Handler(2) | Length(2) |
